# Undersök lead {#investigate-lead}

Upptäck varför någon inte har kvalificerat sig för ett program eller gått igenom livscykeln.

Klicka på pilen till höger om du vill visa en fullständig lista över undersökningsscenarier.

SCREENSHOT

| **Titel** | Beskrivning |
|----------|------------|
| **MQL-undersökning** | Innehåller information om varför ett specifikt lead antingen inte kvalificerades, slutade engagera eller fastnade. |
| **Programkvalificering** | Beskriver varför en lead inte var berättigad till ett vårdprogram. |
| **Livscykelprogression** | Används för att utforska varför en lead fastnar på en scen längre än förväntat. |
| **Förlovningsanalys** | Ska användas när en lead har slutat öppna eller klicka på e-postmeddelanden. |
| **Kampanjresa** | Används för att visa en Leads fullständiga bana och kontaktytor i resan. |
| **Datakvalitet** | Ska användas när det finns en möjlig förväntan att felaktiga data orsakar ett problem med lead-instansen. |
| **Försäljningsleverans** | Ska användas när Försäljning avvisade en lead och inte följt upp. |
| **Attribution** | Används för att utforska vad som påverkade en lead som ledde till att en möjlighet skapades. |
| **Poängminskning** | För att ta reda på varför ett leadpoäng oväntat sjönk. |

>[!TIP]
>
>Om du snabbt vill visa en lista över alla nio scenarier skriver du `/lead investigation` i promptfönstret högst upp.
