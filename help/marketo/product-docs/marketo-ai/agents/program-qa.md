# Program-QA {#program-qa}

Granska dina program för bästa praxis i alla komponenter, som e-post, landningssidor, kampanjer med mera.

Det finns två sätt att få åtkomst till den här agenten.

Klicka på agenten från mittfönstret och klicka på listrutan högst upp till höger

klicka på knappen Använd denna agent

och standardtexten visas i den vänstra rutan i dialogrutan.

Välj sedan det program som du vill ska vara QA i den högra rutan

En sammanfattning av programmet visas i mittfönstret med en översikt över programmet och vad som skapades för det.

Du kan också klicka på agenten på AI-skärmen.

AI-assistenten visar standardtexten som visar vilken agent som kommer att köras.

Välj sedan det program till höger som ska vara QA&#39;d.

och samma sammanfattning visas i mittkonsolen
