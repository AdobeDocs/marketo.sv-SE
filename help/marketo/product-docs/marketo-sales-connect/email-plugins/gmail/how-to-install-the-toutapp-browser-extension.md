# Installera ToutApp Browser Extension {#how-to-install-the-toutapp-browser-extension}

1. Klicka på **Inställningar** i [webbprogrammet](https://toutapp.com/next#settings).

1. Klicka på **E-postinställningar** under Mitt konto.

1. Klicka på **Plugins**.

1. Klicka på **Installera** under Gmail- och Google-appar.
