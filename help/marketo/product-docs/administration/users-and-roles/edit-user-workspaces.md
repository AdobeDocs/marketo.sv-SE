# Redigera användararbetsytor {#edit-user-workspaces}

1. Gå till området **[!UICONTROL Admin]**.

   ![](assets/edit-user-workspaces-1.png)

1. Klicka på **[!UICONTROL Users & Roles]**.

   ![](assets/edit-user-workspaces-2.png)

1. Markera önskad användare och klicka på **[!UICONTROL Edit User]**.

   ![](assets/edit-user-workspaces-3.png)

1. Gör önskade ändringar och klicka på **[!UICONTROL Save]**.

   ![](assets/edit-user-workspaces-4.png)
