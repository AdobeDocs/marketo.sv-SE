# Översikt över MCP-dokument {#overview}

ADERS TEXT:

Marketo Engage publika MCP-server är en molnbaserad tjänst som gör Marketo REST API:er tillgängliga för AI-agenter som strukturerade verktyg. MCP (Model Context Protocol) ger ett standardiserat, schemadrivet gränssnitt som gör att AI-systemen kan interagera med Marketo på ett säkert och effektivt sätt.

I stället för att integrera Marketo-slutpunkter i varje program manuellt tillhandahåller MCP-servern en enda AI-inbyggd gateway som gör att MCP-kompatibla agenter kan identifiera tillgängliga åtgärder, förstå nödvändiga parametrar och utföra åtgärder via ett konsekvent protokoll.

Det finns en mängd tillgängliga verktyg som omfattar både offentliga REST API:er och interna API:er, som utför Marketo kärnfunktioner som i gränssnittet men via API-anrop. Fler verktyg kommer att läggas till kontinuerligt när verktygen utvecklas. &lt;— NEED CLARIFICATION

MCP-servern erbjuds som självbetjäning och kan konfigureras via Adobe IO-ramverket genom att bygga en slutpunkt och autentisera anslutningen. Mer information och installationsinformation finns på Adobe Developer webbplats (lägg till länk här). DEN HÄR? https://developer.adobe.com/marketo-apis/

Kampanjverktyg som visas som SOAP API:er har konverterats till REST API:er och lagts till.

Följ dessa instruktioner (hyperlänk) när du har konfigurerat den för att visa en lista över tillgängliga verktyg.

Obs! Verktyg som inte är tillgängliga är&quot;Inaktivera kampanj&quot; och&quot;Ta bort statisk lista&quot; (TBD).
