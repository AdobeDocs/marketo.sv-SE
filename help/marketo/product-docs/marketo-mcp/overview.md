# Översikt över MCP-dokument {#overview}

ADERS TEXT:

Public MCP Server for Marketo Engage är en molntillgänglig tjänst som exponerar Marketo Engage offentliga REST API:er som strukturerade, identifierbara verktyg för AI-agenter. Det bygger på Model Context Protocol (MCP) och omvandlar Marketo funktioner för automatiserad marknadsföring till ett standardiserat, schemadrivet gränssnitt som AI-systemen kan använda på ett säkert och intelligent sätt.

I stället för att integrera Marketo-slutpunkter i varje program manuellt fungerar MCP-servern som en enda AInative-gateway som gör att MCP-kompatibla agenter kan identifiera tillgängliga åtgärder, förstå nödvändiga parametrar och utföra åtgärder med ett konsekvent protokoll.

Det finns en mängd tillgängliga verktyg som omfattar både offentliga REST API:er och interna API:er, som utför Marketo kärnfunktioner som i gränssnittet men via API-anrop. Fler verktyg kommer att läggas till kontinuerligt när verktygen utvecklas.

MCP-servern erbjuds som självbetjäning och kan konfigureras via Adobe IO-ramverket genom att bygga en slutpunkt och autentisera anslutningen. Mer information och installationsinformation finns på Adobe Developer webbplats på (lägg till länk här).

Kampanjverktyg som visas som SOAP API:er har konverterats till REST API:er och lagts till.

Följ dessa instruktioner (hyperlänk) när du har konfigurerat den för att visa en lista över tillgängliga verktyg.
