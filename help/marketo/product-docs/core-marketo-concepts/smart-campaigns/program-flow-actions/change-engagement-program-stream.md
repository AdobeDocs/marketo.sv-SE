# Change Engagement Program Stream {#change-engagement-program-stream}

## Översikt {#overview}

Normalt använder du [regler för direktövergång](/help/marketo/product-docs/email-marketing/drip-nurturing/engagement-program-streams/transition-people-between-engagement-streams.md) för att uppnå detta, men om du vill flytta personer manuellt från en ström till en annan, är detta det flödessteg som ska användas.

![](assets/image2014-9-22-14-3a52-3a14.png)

## Användning {#usage}

1. Välj det engagemangsprogram som du vill flytta personen till.

   >[!NOTE]
   >
   >Om du väljer ett annat program lämnas personen kvar i den aktuella strömmen och läggs till i den nya.

   ![](assets/image2014-9-22-14-3a52-3a50.png)

1. Markera den ström du vill lägga till dina personer i.

   ![](assets/image2014-9-22-14-3a52-3a59.png)

Och det är allt!
