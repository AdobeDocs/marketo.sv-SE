# Marketo Skyar

Här nedan hittar du det som finns i [!DNL Marketo Sky].

* Omdesignat navigeringsträd

   * Ikonografi
* Global sökning
* Etiketter

## Personalisering

* Mina Marketo-widgetar
   * Förfaller snart
   * Navigering

* Mina anpassade Marketo-vyer

* Smarta kampanjer
   * Sparade regler
   * Sparade flöden
* Engagement Programs
* E-postprogram

* Händelseprogram

* Programmappar
* Kampanjmappar (inte fliken Resultat)
* Sidor med resursinformation
   * E-post
   * Landningssida
   * Formulär
   * Statisk lista
   * Smart List
* Tokens
   * Tokenmappar
   * Länktoken
   * Importera variabler
