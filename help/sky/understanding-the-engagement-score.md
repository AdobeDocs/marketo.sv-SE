# Så här fungerar engagemanget

<br> 

Det blir enkelt att se hur effektivt innehållet i engagemangsprogrammet är. Poängen varierar mellan 0 och 100. Gå [till Engagement Dashboard](https://docs.marketo.com/display/DOCS/The+Engagement+Dashboard) för att se hur du kan spåra innehållets prestanda.

![Bild ett](/help/sky/assets/engagement-programs/understanding-the-engagement-score/understanding-the-engagement-score-1.png)

![Bild två](/help/sky/assets/engagement-programs/understanding-the-engagement-score/understanding-the-engagement-score-2.png)

Poängen baseras på en egen algoritm som tar hänsyn till engagerat beteende (Open, Click, Program Success) och inaktiverat beteende (Unsubscribe). Den är baserad på e-post med dropp- och vårdstil, vilket ger ett genomsnitt på 50. För att ge användarna möjlighet att interagera med ert innehåll beräknas engagemangspoängen 72 timmar efter varje sändning.

>[!NOTE]
>
>När program används som innehåll i strömmar baseras poängen på programmedlemskap och framgångsstatus, inte på e-postinteraktion (klickningar, öppningar, avslut).
>
>Engagemangsmoment är universella för alla kunder. Du kan jämföra dem för att se vem som har det mest engagerande innehållet.

>[!NOTE]
>
>Den egna algoritmen gäller också för engagemangspoängen i e-postprogram.
