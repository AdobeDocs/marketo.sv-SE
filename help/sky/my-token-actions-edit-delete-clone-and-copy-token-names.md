# Mina tokenåtgärder: Redigera, ta bort, klona och kopiera tokennamn

## Redigera en token

Klicka på [!UICONTROL My Tokens] värdet [!UICONTROL **för den token som du vill redigera på fliken**] i kampanjmappen eller -programmet. Redigeringsfönstret öppnas.

![Bild ett](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-1.png)

>[!NOTE]
>
>Textvariabler har inget separat redigeringsfönster. Klicka på [!UICONTROL **Värde**] för att göra textbundna redigeringar.

![Bild två](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-2.png)

## Ta bort en token

1. Högerklicka på variabeln och välj sedan [!UICONTROL **Ta bort**].

   ![Bild tre](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-3.png)

1. Klicka på [!UICONTROL **Bekräfta**].

   ![Bild fyra](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-4.png)

>[!NOTE]
>
>Mina token som för närvarande används i resurser kan inte tas bort. Om du försöker ta bort dem får du ett felmeddelande som visar var variabeln används. Ta bort token från dessa resurser och försök ta bort den igen.

## Klona en token

1. Om du vill klona en Min token högerklickar du på den och väljer [!UICONTROL **Klona**].

   ![Bild fem](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-5.png)

1. Ange ett nytt namn för din klontoken, redigera vid behov och klicka på [!UICONTROL **Spara**].

   ![Bild sex](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-6.png)

## Kopiera ett tokennamn

Om du vill kopiera namnet på en token till Urklipp (d.v.s. `{{my.Event Calendar File}}`) högerklickar du på variabeln och väljer **Kopiera tokennamn**.

![Bild sju](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-7.png)
